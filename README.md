# Hunter x Hunter Shopping App | Using React/Vite 

A simple hard-coded shopping application built using **React** and **Vite**.
This app allows users to browse a selection of Hunter x Hunter themed products (such as T-shirts, toys, and lunchboxes) and purchase items.


# Features

- Displays a list of hard-coded products
- Each product shows an image, name, and price
- Users can select an item to "purchase"
- Checkout screen includes a fake credit card form using Luhn algorithm
- Clean, simple UI using React components


# Tech Stack

- **React**
- **Vite**
- **JavaScript (ES6)**
- **CSS**
